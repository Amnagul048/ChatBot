# Pro Chatbot - C# Desktop Application

![Chatbot Preview](https://img.shields.io/badge/Status-Completed-brightgreen) ![C#](https://img.shields.io/badge/Language-C%23-blue) ![Windows Forms](https://img.shields.io/badge/Framework-WinForms-orange)

## Project Overview
**Pro Chatbot** is a desktop-based chatbot application built in **C#** using **Windows Forms**.  
It simulates a basic conversational AI, capable of responding to user greetings, time/date requests, and basic personal interactions—all **without using external APIs**.  
This project focuses on UI/UX design, message alignment, emoji support, and user-friendly interaction.

---

## Features

- ✅ Modern Dark-themed UI
- ✅ RichTextBox chat display with **timestamps**
- ✅ User message alignment (Right) and Bot messages alignment (Left)
- ✅ Emoji support in bot responses
- ✅ Time and Date queries
- ✅ Personalized greetings (`My name is ...`)
- ✅ Clear Chat button to reset conversation
- ✅ Enter key support for sending messages
